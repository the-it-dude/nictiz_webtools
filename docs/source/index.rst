Welcome to Nictiz Webtools API's documentation!
===============================================

This is Django powered project aimed to serve as API and Django Admin for `Nictiz Mapping Tool <https://mapping.nictiz.deitdude.nl>`_.

Project dependencies are managed by `pip-tools <https://github.com/jazzband/pip-tools>`_, more information can be found in :doc:`managing_dependencies`.

Installation instructions and basic development workflow can be found in :doc:usage.

.. note::

   This project is under active development.
